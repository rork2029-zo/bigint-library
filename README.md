Ntro library provides calculating for big integers.
Addition, Substraction, Multiplication, Division, Squaring and Expoenetiation is main functions.

[1] Compile
After compiling, we can use some features.

====== Command ====================================================================
|    $ make / make all    ---->  Make library, build calculator and verifying	  |
|    $ make ntro    	  ---->  Make library						              |
|    $ make cal           ---->	 Build calculator program 				          |
|	 $ ./cal		      ---->  Start calculator					              |
|    $ make verify 		  ---->  Build verifying functions		         		  |
|	 $ ./verify		      ---->  Start verification for uncommented functions     |
|    $ make clear 		  ---->  Remove all executable files			          |
===================================================================================
